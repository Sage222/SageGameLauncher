# SageGameLauncher
Nice python GUI to launch games from from.  Gets release date and image from steam. Can reorder and mark as favourite. Can use a controller or mouse and keyboard. Enjoy.

Needs python 3.12 for pygame to work. pygame does not support newer at this stage.

Run:
python -m pip install PyQt6 requests pygame pywin32

then python sagegamelauncherXX.py.

(where XX is the version).

put the placeholder.png in the same directory as the script for games where it cannot find the correct image from steam.

use right click menu on games to manage them.

Enjoy
